BadBot Administration Commands list

b!{Command} This is what you would use to talk with BadBot.

### Simply non-destructive.

Simply non-destructive is something everyone could use as it doesn't break the server.

- admin
Sends you a DM of the link you're visiting right now!
- badbot
Sends a message on the current channel!

### Destructive DONT USE Unless you want to kill the server
These are CAPS SENSITIVE TO PREVENT ACCIDENTAL COMMAND DROPPING
- DC
Deletes the current channel no matter what is on it.
- BULKCLEAR
Deletes all messages on the channel selected
